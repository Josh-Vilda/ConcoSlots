# Raspberry Pi 5 Slot Machine

Python 3 slot machine game for Raspberry Pi 5 using pygame and GPIO-based crank/button controls. Runs fullscreen with free infinite spins and a desktop keyboard fallback.

## Requirements

- Python 3
- Raspberry Pi OS (for GPIO mode) or any desktop OS (fallback)
- `pygame`, `gpiozero`

Install:

```bash
pip install -r requirements.txt
